# kebron

ALX software engineering program project

Kebron Rents is A platform Based on Ethiopia Which is Online Home/Appartment Renting System.

# Frameworks :

       React
       Tailwind
       Django
       Posgresql

# API Documentation

http://localhost:8000/api/v1/docs

# Back-end setup

    cd back-end/
    python3 manage.py runserver

# Front-end setup

    cd front-end/
    npm install
    npm run dev
